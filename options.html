Getting Started

Tab Previews

Quickly switch between your most recently used tabs with live screenshot previews. The switcher shows your tabs ordered by last use, so your most relevant tabs are always one shortcut away.

Previews are captured automatically as you browse. Tabs you haven't visited yet will show a placeholder until you switch to them.

Setup Required: Keyboard Shortcuts

To use the tab switcher, you need to set a keyboard shortcut in Chrome. The extension doesn't have a default shortcut enabled by Chrome — you must assign one manually:

  1. Click the button below or go to chrome://extensions/shortcuts
  2. Find Tab Switcher Previews
  3. Set a shortcut for "Show tab switcher and go forward" (e.g. Ctrl+Tab)
  4. Optionally set one for "Show tab switcher and go backward"
